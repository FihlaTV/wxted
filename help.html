Special keys
Page Up - Show next page of carousel
Page Down - Show previous page of carousel
F11 - Hide/Reveal
CTRL-Z - Undo
Alt-Gr - Show control codes
Text colours
Shift-F1 - Red text
Shift-F2 - Green text
Shift-F3 - Yellow text
Shift-F4 - Blue text
Shift-F5 - Magenta text
Shift-F6 - Cyan text
Shift-F7 - White text
 Graphic colours When the small graphics cursor is displayed then pressing space will toggle the pixel on and off.
Ctrl-F1 - Graphics red
Ctrl-F2 - Graphics green
Ctrl-F3 - Graphics yellow
Ctrl-F4 - Graphics blue
Ctrl-F5 - Graphics magenta
Ctrl-F6 - Graphics cyan
Ctrl-F7 - Graphics white
Effects
CTRL-H - Flash
CTRL-I - Steady
CTRL-J - End Box
CTRL-K - Start Box
CTRL-L - Normal height
CTRL-M (enter) - Double height
CTRL-R - Conceal display
CTRL-Y - Contiguous graphics
CTRL-T - Separated graphics
CTRL-U - Black background
CTRL-B - New background
CTRL-W - Hold graphics
CTRL-X - Release graphics
